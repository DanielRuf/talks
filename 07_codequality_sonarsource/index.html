SonarSource has different solutions
for improving code quality
  • sonar-scanner CLI (the core)
  • SonarQube (the analysis platform)
  • SonarLint (linting in IDE, requires JDK)
  • SonarCloud (hosted version of SonarQube)
install sonar-scanner
  • extract archive anywhere
  • add path to bin directory to your PATH variable
  • sonar-scanner --help
  • (alternatively using sonarqube-scanner (NodeJS))
install SonarQube demo instance
  • Docker (docker pull sonarqube)
  • or manually
SonarQube should run on port 9000 by default
(docker run -p 9000:9000 sonarqube)
setup a new project
start SonarQube
create config file

projectKey is required, everything else is optional
add .scannerwork directory (project cache) to .gitignore
run sonar-scanner in project
check results
demo login is admin / admin
many default rules and rule sets
